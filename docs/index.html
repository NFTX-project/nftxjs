NFTX.js

This is an SDK that exposes all of the core functionality of NFTX:
  • Searching and fetching vaults
  • Fetching detailed vault and pool info
  • Fetching position details
  • Getting vault price quotes
  • Buying
  • Selling
  • Swapping
  • Staking

Getting started

Installation

        
          npm install nftx.js

Usage

        
          import { buy, configure, fetchVault } from 'nftx.js';

          configure({
            network: 1,
            keys: {
              NFTX_API: '<>'
            }
          });

          const vault = await fetchVault({ vaultId: '0' });

          const transaction = await buy({
            signer,
            userAddress: '0x0000',
            vault,
            tokenIds: ['2712'],
            randomBuys: 1
          });

          const receipt = await transaction.wait();
